Politique de confidentialité - PeekDrive
Dernière mise à jour : 27 mars 2026
**Ce que fait PeekDrive**
PeekDrive est une extension Chrome qui vous permet de rechercher dans le contenu de vos fichiers Google Drive et de visualiser les passages correspondants directement depuis les résultats.
Collecte de données
PeekDrive ne collecte, ne stocke, ne transmet et ne partage aucune donnée personnelle. Tout le traitement se fait localement dans votre navigateur. Vos recherches, le contenu de vos fichiers et vos résultats ne quittent jamais votre appareil.
Plus précisément, PeekDrive :

N'envoie aucune donnée vers des serveurs externes
N'utilise aucun outil d'analyse, de suivi ou de télémétrie
N'utilise pas de cookies
Ne stocke ni le contenu de vos fichiers ni votre historique de recherche
Ne partage aucune donnée avec des tiers

**Utilisation des API Google**
PeekDrive accède à vos fichiers Google Drive via les API de Google uniquement pour effectuer la recherche et afficher les passages correspondants. Cet accès est limité à ce qui est nécessaire au fonctionnement de l'extension.
L'utilisation des informations reçues des API Google par PeekDrive est conforme à la politique de données utilisateur du Chrome Web Store, y compris les exigences d'utilisation limitée (Limited Use).
Autorisations
PeekDrive demande l'accès à votre Google Drive uniquement pour lire le contenu des fichiers à des fins de recherche. Aucune autre autorisation n'est demandée au-delà de ce qui est strictement nécessaire.
Modifications
En cas de modification de cette politique, la version mise à jour sera publiée à cette même URL avec une nouvelle date.
