How PeekDrive works

Search file content

Type a word or phrase. PeekDrive scans the actual text inside your Google Drive files, not just file names.

Preview the passage

See the matching text in context, right in the results, without opening the file.

Jump to the right spot

Click "Open" and land on the exact paragraph, cell, or slide. No more Ctrl+F inside every file.

Mini-table for spreadsheets

For Sheets and Excel, PeekDrive shows a compact preview with the matching row and column.

Filter and sort

Narrow results by file type (Docs, Sheets, Slides, PDF) and sort by date or relevance.

100% private

All processing happens locally in your browser. No data is sent to external servers. No analytics, no tracking.
